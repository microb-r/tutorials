# Prokaryotic and Eukaryotic Genomes Submission to NCBI

Submit genomes [here](https://submit.ncbi.nlm.nih.gov/subs/genome/)

1. Click on **New Submission**

2. In the case of **Single BioSample**:
  
2.1 Select ‘Single genome’ and continue.

2.2. Add/update information of submitter.
  
<img width="1412" alt="Screen Shot 2019-08-18 at 7 32 37 PM" src="https://user-images.githubusercontent.com/48694025/63231804-fba85100-c1ee-11e9-8cf3-884d36785963.png">
  
2.3. Fill **General Information** with BioProject number (previously create a Bioproject and associated BioSample). Select date of release.
  
<img width="1409" alt="Screen Shot 2019-08-18 at 7 47 55 PM" src="https://user-images.githubusercontent.com/48694025/63231974-167bc500-c1f1-11e9-9432-8d4798c98aaf.png">

2.4 Fill **Genome Info** section according to image below, including **Submission title**. Continue.

2. In the case of ‘Batch Submission’:
	2.2. Select Batch/multiple genomes
	2.3. Add/update information of submitter.
	2.4. Fill ‘General Information’ with BioProject number (previously create a Bioproject and associated BioSample). Select date of release.
	2.5. In ‘Gaps Section’ check No for the first question (not randomly assembled). Fill the rest of the information as indicated below. Continue.
	2.6 In ‘Genome Info’ section, select the WGS option. Fill the rest of the information as indicated below. 
	2.7 Download template excel and fill with genome’s information. Save tab as .txt file and upload.
	2.8 Upload either .fasta or .sqn files. .sqn files can be created from genbank files here https://chlorobox.mpimp-golm.mpg.de/GenBank2Sequin.html#.
	2.9 Review information and submit.
